# Thiago Lopes Machado — Portfólio Pessoal

Primeira versão do meu portfólio pessoal desenvolvido com foco em performance, responsividade e organização de código utilizando tecnologias modernas do desenvolvimento web.

## 📌 Sobre o Projeto

Este projeto representa a versão `1.0` do meu portfólio como desenvolvedor Full Stack.

O objetivo principal é apresentar minhas habilidades, evolução no desenvolvimento web e servir como base para futuras melhorias e novas tecnologias.

O projeto foi desenvolvido utilizando JavaScript puro, HTML5 e CSS3, mantendo uma arquitetura simples, organizada e escalável.

---

## 🚀 Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Vite
- Git & GitHub

## ▶️ Como Executar o Projeto

### Clone o repositório

```bash
git clone https://github.com/GitBrowserBit/portifolio-pessoal.git
```

### Acesse a pasta do projeto

```bash
cd portifolio-pessoal
```

### Instale as dependências

```bash
npm install
```

### Execute o servidor de desenvolvimento

```bash
npm run dev
```

Após executar o comando, o Vite irá gerar um endereço local semelhante a:

```txt
http://localhost:5173
```

Abra o endereço no navegador para visualizar o projeto.
